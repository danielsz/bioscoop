* Description

Bioscoop gives you a macro that transforms code like this:

#+BEGIN_SRC clojure
(let [logbook (get-logbook entity)]
  (when (some pred logbook)
    (let [response (body)]
      (when (success? response)
        (write logbook)))))
#+END_SRC
